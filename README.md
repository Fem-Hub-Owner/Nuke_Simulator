# ☢️ Nuclear Test Simulator

A cinematic 3D nuclear explosion simulator built with **HTML, CSS, JavaScript, and Three.js**.

Experience a full-screen explosion scene with particles, smoke, lighting effects, camera shake, and cinematic visuals.

---

## 🚀 Features

- ☢️ Nuclear detonation button
- 🔥 Expanding fireball
- 💥 Debris and particle physics
- 🍄 Mushroom cloud smoke system
- 🌊 Shockwave effect
- 🌡 Heat distortion effect
- 📸 Camera shake
- ✨ Bloom lighting effects
- 🌎 3D environment
- 🔊 Explosion sound system

- HTML5
- CSS3
- JavaScript
- Three.js
